# Koibito 交友平台 — 產品需求文件（PRD）

**版本**：v1.1
**日期**：2026-03-26
**目標市場**：台灣（繁體中文主介面）
**平台**：iOS / Android App（Web 參考）

---

## 1. 產品概述

Koibito 是一款面向台灣市場的交友平台，核心機制圍繞「配對探索 → 互動破冰 → 深度聊天」三階段設計。平台採用男女差異化定價策略：女性用戶享有較高的免費權益，男性用戶透過購買會籍（VIP）及虛擬道具解鎖核心互動功能。

---

## 2. 目標用戶

| 面向 | 說明 |
|---|---|
| 主要市場 | 台灣 20–40 歲單身用戶 |
| 語言 | 繁體中文為主，輔助日文、英文、簡體中文 |
| 付款偏好 | 信用卡、超商代碼、Web3（USDT） |
| 性別策略 | 女性免費/優惠吸引，男性付費驅動核心功能 |

---

## 3. 用戶角色定義

| 角色 | 說明 |
|---|---|
| 一般用戶（未付費男性） | 可瀏覽、篩選，但無法主動發起聊天 |
| VIP 會員（付費男性） | 解鎖聊天、特殊配對權益 |
| 女性用戶 | 免費使用大部分功能，享有每日閃電罐頭免費配額 |
| 特殊角色 | 享有特殊反傭比例（後台設定） |

## 4.6 閃電罐頭

閃電罐頭為「批量破冰」功能，在指定時間內向符合篩選條件的用戶自動發送打招呼訊息。

| 項目 | 說明 |
|---|---|
| 女性用戶 | 每天免費開啟一次 |
| 男性用戶 | 需購買後使用 |
| 特殊權限 | 享有每日自動發送功能 |
| 發送條件 | 依篩選條件設定目標用戶範圍 |
| 打招呼內容 | 自訂文字 |
| 倒數計時 | 顯示本次發送剩餘時間 |
